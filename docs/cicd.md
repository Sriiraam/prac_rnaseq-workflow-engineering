# Phase 9 — CI/CD

## Goal

Automate testing of the RNA-seq workflow using GitHub Actions.

## Steps Completed

* Created `.github/workflows/ci.yml`
* Configured GitHub Actions
* Installed Java in CI runner

* Verified workflow execution automatically on every push

* Continuous Integration successfully configured
* Pipeline validation now automated
* Repository became production-style and reproducible
