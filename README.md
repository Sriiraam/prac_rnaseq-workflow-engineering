# RNA-seq Workflow Engineering

![CI](https://github.com/Sriiraam/prac_rnaseq-workflow-engineering/actions/workflows/ci.yml/badge.svg)# prac_RNA-seq Workflow Engineering

## Objective

Build a reproducible RNA-seq workflow using:

- Nextflow DSL2
- Docker
- AWS
- GitHub Actions

## Pipeline Stages

1. Quality Control
2. Trimming
3. Alignment
4. Quantification
5. Differential Expression Analysis
6. Reporting

## Engineering Goals

- Reproducibility
- Scalability
- Automation
- Cloud Execution
- CI/CD
